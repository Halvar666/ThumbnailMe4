MediaInfo / MediaInfoLib notices
=================================

Thumbnail me 4 dynamically loads MediaInfo when its DLL is present.
If MediaInfo is not present, the application uses FFmpeg fallback metadata.

Required binary redistribution notice:

  This product uses MediaInfo library, Copyright (c) 2002-2026
  MediaArea.net SARL.

MediaInfoLib redistribution summary:
- Source and binary redistribution are permitted.
- Source redistribution must keep the copyright notice, license conditions
  and disclaimer.
- Binary redistribution must reproduce the copyright notice, license
  conditions and disclaimer in documentation and/or materials.
- MediaInfo also offers the short binary redistribution sentence above.

Keep this file in Windows ZIP binary packages that include MediaInfo DLLs.
